================================================================================
                    v15_cpp Manual Build - Quick Start
================================================================================

NO CMAKE REQUIRED! This is a self-contained build system.

--------------------------------------------------------------------------------
STEP 1: Test Your Environment (2 seconds)
--------------------------------------------------------------------------------

    ./test_build.sh

Expected output:
    ✓ Compiler detected
    ✓ C++17 support verified
    ✓ Eigen3 found or will be downloaded
    ✓ Test files compiled successfully

--------------------------------------------------------------------------------
STEP 2: Build the Project (15 seconds)
--------------------------------------------------------------------------------

    ./build_manual.sh

What it does:
    • Detects clang++ or g++
    • Downloads Eigen3 if needed (header-only, ~40MB)
    • Compiles all .cpp files in src/
    • Creates libv15scanner.a static library
    • Builds executables and tests

Output directory: build_manual/

--------------------------------------------------------------------------------
STEP 3: Use the Library
--------------------------------------------------------------------------------

See how to link your programs:

    ./example_link.sh

Basic template:

    clang++ -std=c++17 -O3 \
        -I./include \
        -I./build_manual/deps/eigen3 \
        your_program.cpp \
        -L./build_manual/lib -lv15scanner \
        -o your_program

--------------------------------------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------------------------------------

If build fails with errors about "duplicate Channel" or "no default constructor":

    ./fix_build_errors.sh
    ./build_manual.sh

These are code issues in the existing codebase, not build system issues.

REQUIREMENTS
--------------------------------------------------------------------------------

    ✓ clang++ or g++ with C++17 support
    ✓ curl or wget (for downloading Eigen)

    ✗ No package managers needed
    ✗ No sudo/root needed
    ✗ No Python needed (for C++ build)
