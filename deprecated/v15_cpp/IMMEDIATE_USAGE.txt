================================================================================
                         TROUBLESHOOTING
================================================================================

If scanner doesn't run:
  1. Check data directory exists: ls ../data
  2. Check executable exists: ls build/v15_scanner
  3. Rebuild if needed: ./build_manual.sh
  4. Check permissions: chmod +x build/v15_scanner

If samples = 0:
  1. Add --verbose flag to see details
  2. Check Pass 2 "Valid labels" count > 0
  3. Ensure dataset has >60,000 bars (warmup requirement)
  4. Try smaller --step value (more channels)

If performance is slow:
  1. Use --workers 8 (or your CPU core count)
  2. Use SSD for data directory (not HDD)
  3. Close other applications (free up RAM)
  4. Check --verbose for bottlenecks
